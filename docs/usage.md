this project can be used as a template to deploy various vms using vagrant in different virtualizations software such vmware fusion and parallels.


# VMWARE FUSION
running vagrant and vmware fusion can be tricky sometimes specially due to incompatible components, here are a list of components and tested versions.

components|tested and supported version
--------|----------------------------
vagrant | 2.4.8
vagrant-vmware-desktop| 3.0.5
vmware fusion pro | 13.6.4 (24832108)
vagrant vmware utility | 1.0.24
